Day01

一、DDL之库的操作

1.查看所有的库

show databases ;

2.创建库

常见的字符集：UTF9 GBK GB2312 GB18030

create database mydb charset =UTF8;

create database if not exists mydb charset =UTF8;


3.查看创建库的语句【字符集】

show create database mydb;


4.使用库【切换库】

use mydb;

5.删除库

drop database mydb;

drop database if exists mydb;


二、表的管理

一定要先切换库，再创建表

show databases;

create database if not exists mydb charset =UTF8;

use mydb;

1.创建表【指定字段名和数据类型 int代表整形，char(n)代表字符串类型】

create table if not exists t1(Id int,name char(20),age int);

2.查看当前库中的所有表

show tables;

3.查看创建表的语句【字符集】

show create table t1;

4.查看表结构

desc t1;

5.删除表

drop table if exists t1;
